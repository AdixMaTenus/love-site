## 🔄 Какие изменения происходят при загрузке страницы

```
1. HTML загружается
   ↓
2. script.js парсится и выполняется
   ↓
3. IMAGE_LOADER.init() запускается
   ↓
4. Система определяет: production или localhost?
   ↓
5. Для каждого .gallery-photo:
   a. Получает исходный src (например "public/utro.jpg")
   b. Создает массив вариантов путей
   c. Попробует загрузить первый вариант
   d. Если не удаче → следующий вариант
   e. После исчерпания вариантов → retry с задержкой
   ↓
6. Все логи выводятся в Console
   ↓
7. Фотографии загружаются успешно! ✅
```
